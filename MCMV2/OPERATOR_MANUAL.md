# Инструкция для оператора: MCMV2

## Что это за программа

`MCMV2` - это программа для запуска готового сценария испытания на машине.

Простыми словами:

- программа открывает файл сценария Excel;
- отправляет команды в контроллер по COM-порту;
- показывает давления и токи на экране;
- записывает результаты в файл;
- потом позволяет открыть графики.

## Что нужно перед началом работы

Перед запуском программы проверьте:

- компьютер включен;
- контроллер подключен к компьютеру;
- известен нужный `COM`-порт;
- есть готовый файл сценария Excel;
- оборудование находится в безопасном состоянии;
- запущена только одна копия программы `MCMV2`.

## Какие файлы использует программа

- Сценарий: файл Excel `.xls`
- Результат испытания: файл `.txt`
- Отчеты и результаты обычно лежат в папке пользователя:
  `%USERPROFILE%\mcm\reports\`

Важно:

- для загрузки сценария в `MCMV2` используйте именно `.xls`;
- если сценарий делали в `WizardScena`, сохраняйте его как `.xls`.

## Основные кнопки на стартовом экране

- `Open Scenario` - открыть новый файл сценария
- `Open last scenario` - открыть последний сценарий, который использовался ранее
- `Open Chart` - открыть окно просмотра графиков
- `Settings` - настройки COM-порта, скорости и режима экрана

## Как обычно работать

### 1. Открыть сценарий

1. Запустите `MCMV2`.
2. Нажмите `Open Scenario`.
3. Выберите нужный файл Excel `.xls`.
4. Дождитесь загрузки сценария.
5. Если сценарий загрузился успешно, откроется основной экран.

### 2. Открыть последний сценарий

1. Запустите `MCMV2`.
2. Нажмите `Open last scenario`.
3. Если файл еще существует и не поврежден, программа откроет его сразу.

### 3. Проверить настройки связи

1. На стартовом экране нажмите `Settings`.
2. Найдите строку `COM Port`.
3. Выберите нужный порт из списка.
4. Проверьте `Baud-rate`.
5. Закройте настройки или вернитесь к основным кнопкам.

Если нужного COM-порта нет:

- проверьте кабель;
- проверьте питание контроллера;
- переподключите устройство;
- закройте и снова откройте программу.

### 4. Запустить испытание

1. Откройте сценарий.
2. Убедитесь, что на основном экране видны данные и нужные каналы.
3. Убедитесь, что оборудование готово к работе.
4. Нажмите кнопку `Play`.
5. Дождитесь окончания сценария или остановите процесс по внутреннему регламенту.

Важно:

- не нажимайте `Play` повторно без необходимости;
- перед запуском убедитесь, что подключение к оборудованию выполнено правильно.

### 5. Посмотреть график

1. После испытания откройте `Open Chart`.
2. Выберите нужный файл графика, если программа не открыла его автоматически.
3. Сравните результат с эталоном, если это требуется.

## Что показывает основной экран

На основном экране обычно видно:

- давления;
- токи;
- состояние каналов;
- кнопки управления испытанием;
- элементы просмотра графиков.

Если оператору нужно только запустить готовый сценарий, обычно достаточно:

- открыть сценарий;
- проверить COM;
- нажать `Play`;
- сохранить или открыть результат потом.

## Типовые алгоритмы

### Алгоритм 1. Запуск нового испытания

1. Подключить контроллер.
2. Запустить `MCMV2`.
3. Открыть `Settings` и проверить `COM Port`.
4. Нажать `Open Scenario`.
5. Выбрать файл `.xls`.
6. Дождаться открытия основного экрана.
7. Проверить готовность установки.
8. Нажать `Play`.
9. После завершения открыть график или найти результат в папке отчетов.

### Алгоритм 2. Повторный запуск последнего сценария

1. Запустить `MCMV2`.
2. Нажать `Open last scenario`.
3. Проверить, что загрузился нужный сценарий.
4. Проверить COM-порт.
5. Нажать `Play`.

### Алгоритм 3. Просмотр результата после испытания

1. Запустить `MCMV2`.
2. Нажать `Open Chart`.
3. Выбрать нужный файл результата `.txt`, если программа просит.
4. Посмотреть линии и сравнить результат.

### Алгоритм 4. Если сценарий не открывается

1. Убедиться, что выбран правильный файл.
2. Проверить, что это файл Excel `.xls`.
3. Если файл сделан в `WizardScena`, пересохранить его еще раз как `.xls`.
4. Не переименовывать файл вручную из `.xlsx` в `.xls`.
5. Если проблема осталась, передать файл инженеру.

## Частые проблемы

### Программа не видит COM-порт

Что делать:

1. Проверить кабель.
2. Проверить питание устройства.
3. Переподключить устройство.
4. Перезапустить программу.

### Сценарий не загружается

Возможные причины:

- файл не того формата;
- файл поврежден;
- структура Excel неправильная.

Что делать:

1. Взять файл из `WizardScena`.
2. Убедиться, что это `.xls`.
3. Открыть другой сценарий для проверки.

### График не открывается

Что делать:

1. Убедиться, что испытание действительно запускалось.
2. Проверить папку `%USERPROFILE%\mcm\reports\`.
3. Выбрать нужный `.txt` файл вручную.

## Важные правила

- Не редактируйте Excel-файл вручную, если не уверены в своих действиях.
- Не запускайте испытание, если не проверен COM-порт.
- Не работайте сразу в двух копиях `MCMV2`.
- Если есть сомнение в сценарии, сначала откройте его в `WizardScena` и проверьте.

## Очень коротко

Если совсем кратко, обычная работа такая:

1. Запустить `MCMV2`.
2. Проверить `Settings`.
3. Нажать `Open Scenario`.
4. Выбрать `.xls`.
5. Нажать `Play`.
6. После окончания открыть график.
