# Инструкция для оператора: MCMV2

## Для чего нужна программа

`MCMV2` нужна для работы с оборудованием.

Программа умеет:

- открывать готовый сценарий Excel;
- отправлять сценарий в контроллер;
- показывать давления и токи на экране;
- запускать испытание;
- сохранять результат;
- открывать графики результата.

Проще всего запомнить так:

- `WizardScena` - это программа, где сценарий делают;
- `MCMV2` - это программа, где сценарий запускают.

## Что нужно перед началом

Перед работой проверьте:

- компьютер включен;
- контроллер подключен;
- кабель связи подключен надежно;
- известен нужный `COM`-порт;
- есть готовый файл сценария Excel `.xls`;
- оборудование безопасно для запуска.

## Какие файлы нужны

- сценарий для запуска: `.xls`
- результат испытания: `.txt`
- графики и отчеты обычно находятся в:
  `%USERPROFILE%\mcm\reports\`

Важно:

- для `MCMV2` сценарий должен быть именно `.xls`;
- если сценарий сделали в `WizardScena`, его нужно сохранить как `.xls`.

## Что видно на стартовом экране

На стартовом экране обычно есть кнопки:

- `Open Scenario` - открыть новый сценарий
- `Open last scenario` - открыть последний сценарий
- `Open Chart` - открыть графики
- `Settings` - открыть настройки

## Что делать по шагам

### Открыть новый сценарий

1. Запустите `MCMV2`.
2. Нажмите `Open Scenario`.
3. Выберите файл `.xls`.
4. Подождите, пока сценарий загрузится.
5. После загрузки откроется основной экран.

### Открыть последний сценарий

1. Запустите `MCMV2`.
2. Нажмите `Open last scenario`.
3. Проверьте, что открылся нужный файл.

### Проверить настройки связи

1. На стартовом экране нажмите `Settings`.
2. Найдите строку `COM Port`.
3. Выберите правильный порт.
4. Проверьте `Baud-rate`.
5. Если все верно, вернитесь к запуску сценария.

Если COM-порт не виден:

- проверьте кабель;
- проверьте питание устройства;
- переподключите устройство;
- закройте и снова откройте программу.

### Запустить испытание

1. Откройте сценарий.
2. Проверьте, что оборудование готово.
3. Проверьте, что на экране видны каналы и данные.
4. Нажмите `Play`.
5. Дождитесь завершения испытания.

Важно:

- не запускайте испытание, если не проверили COM-порт;
- не нажимайте `Play` много раз подряд.

### Открыть график

1. Нажмите `Open Chart`.
2. Если программа попросит файл, выберите нужный `.txt`.
3. Посмотрите результат и сравните с эталоном, если это требуется.

## Что видно на основном экране

На основном экране оператор обычно видит:

- давления;
- токи;
- состояние каналов;
- кнопки управления;
- элементы для просмотра результата.

Если задача простая, оператору чаще всего достаточно:

1. открыть сценарий;
2. проверить COM;
3. нажать `Play`;
4. потом открыть график.

## Типовые алгоритмы работы

### Алгоритм 1. Обычный запуск испытания

1. Подключить оборудование.
2. Запустить `MCMV2`.
3. Открыть `Settings`.
4. Проверить `COM Port` и `Baud-rate`.
5. Нажать `Open Scenario`.
6. Выбрать файл `.xls`.
7. Проверить готовность установки.
8. Нажать `Play`.
9. После завершения открыть график или передать результат дальше.

### Алгоритм 2. Повторить прошлое испытание

1. Запустить `MCMV2`.
2. Нажать `Open last scenario`.
3. Проверить, что открылся правильный сценарий.
4. Проверить настройки связи.
5. Нажать `Play`.

### Алгоритм 3. Посмотреть старый результат

1. Запустить `MCMV2`.
2. Нажать `Open Chart`.
3. Выбрать нужный `.txt` файл.
4. Проверить график.

### Алгоритм 4. Если сценарий не открывается

1. Убедиться, что выбран правильный файл.
2. Проверить, что это `.xls`.
3. Если файл пришел из `WizardScena`, пересохранить его заново как `.xls`.
4. Не переименовывать `.xlsx` в `.xls` вручную.
5. Если не помогло, передать файл инженеру.

## Частые проблемы

### Программа не видит COM-порт

Что делать:

1. Проверить кабель.
2. Проверить питание контроллера.
3. Переподключить устройство.
4. Перезапустить программу.

### Сценарий не загружается

Возможные причины:

- файл не того формата;
- файл поврежден;
- файл заполнен неправильно.

Что делать:

1. Открыть другой сценарий для проверки.
2. Если этот сценарий делали в `WizardScena`, открыть его там и сохранить снова.
3. Убедиться, что сохранение было в `.xls`.

### График не открывается

Что делать:

1. Убедиться, что испытание действительно запускалось.
2. Проверить папку `%USERPROFILE%\mcm\reports\`.
3. Открыть нужный `.txt` файл вручную.

## Важные правила

- Не запускайте сразу две копии `MCMV2`.
- Не меняйте Excel-файл вручную, если не уверены.
- Не начинайте испытание, пока не проверен COM-порт.
- Если сомневаетесь в файле, сначала проверьте его в `WizardScena`.

## Короткая памятка

1. Запустить `MCMV2`.
2. Проверить `Settings`.
3. Нажать `Open Scenario`.
4. Выбрать `.xls`.
5. Нажать `Play`.
6. После испытания открыть график.
