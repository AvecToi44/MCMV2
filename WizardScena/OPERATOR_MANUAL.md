# Инструкция для оператора: WizardScena

## Что это за программа

`WizardScena` - это программа для создания и редактирования сценариев Excel.

Простыми словами:

- здесь вы делаете сценарий;
- сохраняете его в Excel;
- потом этот Excel открывается в программе `MCMV2`.

То есть:

- `WizardScena` - подготовка сценария;
- `MCMV2` - запуск сценария на оборудовании.

## Что можно делать в программе

- создать новый сценарий;
- открыть готовый сценарий;
- исправить давления;
- исправить токи и соленоиды;
- задать 10 параметров `0x68`;
- сохранить файл Excel;
- выбрать эталонный `.txt` файл для сравнения графиков.

## Какие файлы использует программа

- открыть можно файлы: `.xls`, `.xlsx`
- сохранять для `MCMV2` нужно как: `.xls`
- эталонный файл графика: `.txt`

Важно:

- для программы `MCMV2` нужен именно `.xls`;
- не надо вручную переименовывать `.xlsx` в `.xls`;
- нужно именно сохранить файл через саму программу.

## Что видно на стартовом экране

На стартовом экране есть:

- `Открыть сценарий`
- `Создать новый сценарий`
- `Последние сценарии`
- `Настройки` -> выбор языка

Список последних сценариев показывает 5 последних файлов.

## Как обычно работать

### 1. Создать новый сценарий

1. Запустите `WizardScena`.
2. Нажмите `Создать новый сценарий`.
3. Заполните нужные вкладки.
4. Нажмите `Сохранить Excel`.
5. В окне сохранения задайте имя файла и сохраните файл как `.xls`.

Если имя не введено, программа сама подставит имя `Новый сценарий`.

### 2. Открыть готовый сценарий

1. Запустите `WizardScena`.
2. Нажмите `Открыть сценарий`.
3. Выберите файл `.xls` или `.xlsx`.
4. Дождитесь загрузки.
5. Проверьте данные на вкладках.

### 3. Открыть один из последних сценариев

1. Запустите `WizardScena`.
2. Найдите блок `Последние сценарии`.
3. Нажмите на нужный файл в списке.
4. Сценарий откроется автоматически.

### 4. Сохранить сценарий

1. Нажмите `Сохранить Excel`.
2. Выберите папку.
3. Введите имя файла.
4. Убедитесь, что файл сохраняется как `.xls`.
5. Нажмите `Сохранить`.

## Что означают вкладки

### Вкладка `Основной сценарий`

Здесь задаются шаги сценария:

- текст шага;
- длительность;
- время перехода;
- значения по каналам;
- analog 1;
- analog 2.

Основные кнопки:

- `Добавить шаг`
- `Удалить`
- `Копировать выбранные`
- `Вставить до`
- `Вставить после`
- `Выбрать все`
- `Снять выбор`

### Вкладка `Давления`

Здесь редактируются параметры каналов давления:

- отображаемое имя;
- минимум;
- максимум;
- допуск;
- единица измерения;
- видимость канала.

### Вкладка `Токи`

Здесь редактируются параметры соленоидов:

- главная частота;
- отображаемое имя;
- максимальное значение PWM;
- цена деления;
- амплитуда дитера;
- частота дитера;
- минимальный ток;
- максимальный ток;
- видимость канала.

### Вкладка `Параметры 0x68`

Здесь задаются 10 специальных параметров:

- `Параметр 1`
- `Параметр 2`
- ...
- `Параметр 10`

Эти значения попадают в Excel в ячейки:

- `D14` ... `M14`

Допустимые значения:

- от `0` до `255`

## Верхние кнопки редактора

В редакторе сверху есть кнопки:

- `Выбрать эталон .txt` - выбрать эталонный файл графика
- `Сохранить Excel` - сохранить сценарий
- `Открыть Excel` - открыть другой сценарий
- `Назад на стартовый экран` - вернуться к начальному окну программы
- `Настройки` - смена языка

## Для чего нужен эталонный `.txt`

Эта кнопка нужна, если вы хотите привязать к сценарию эталонный график.

Простыми словами:

- выбирается файл `.txt`;
- путь к нему записывается в Excel;
- потом `MCMV2` может использовать этот эталон при просмотре результатов.

Если эталон не нужен, эту кнопку можно не использовать.

## Типовые алгоритмы

### Алгоритм 1. Создать новый сценарий с нуля

1. Открыть `WizardScena`.
2. Нажать `Создать новый сценарий`.
3. На вкладке `Давления` заполнить названия и диапазоны.
4. На вкладке `Токи` заполнить параметры соленоидов.
5. На вкладке `Параметры 0x68` заполнить 10 параметров, если они нужны.
6. На вкладке `Основной сценарий` добавить шаги.
7. Нажать `Сохранить Excel`.
8. Сохранить файл как `.xls`.
9. Открыть этот файл уже в `MCMV2`.

### Алгоритм 2. Исправить готовый сценарий

1. Открыть `WizardScena`.
2. Нажать `Открыть сценарий`.
3. Выбрать нужный файл.
4. Внести исправления в нужной вкладке.
5. Нажать `Сохранить Excel`.
6. Сохранить файл под тем же именем или новым именем.

### Алгоритм 3. Подготовить сценарий для MCMV2

1. Открыть или создать сценарий в `WizardScena`.
2. Проверить вкладки `Давления`, `Токи`, `Параметры 0x68`, `Основной сценарий`.
3. Если нужен эталон, нажать `Выбрать эталон .txt`.
4. Нажать `Сохранить Excel`.
5. Сохранить именно в формате `.xls`.
6. Затем открыть этот файл в `MCMV2`.

### Алгоритм 4. Быстро открыть последний сценарий

1. Открыть `WizardScena`.
2. На стартовом экране выбрать файл из списка `Последние сценарии`.
3. Проверить данные.
4. При необходимости исправить и сохранить заново.

## Частые проблемы

### Сценарий не открывается

Что делать:

1. Проверить, что файл действительно Excel.
2. Попробовать открыть другой файл.
3. Если файл старый или поврежден, взять резервную копию.

### MCMV2 не принимает сценарий

Что проверить:

1. Сценарий должен быть сохранен как `.xls`.
2. Должны быть заполнены обязательные строки Excel.
3. Параметры `0x68` должны быть в диапазоне `0..255`.
4. Не нужно вручную менять структуру файла Excel.

### Не тот язык интерфейса

Что делать:

1. Нажать `Настройки`.
2. Выбрать `Русский` или `English`.

Программа запомнит язык сама.

## Где программа хранит свои настройки

Программа сама хранит служебные настройки пользователя:

- выбранный язык;
- список последних сценариев.

Оператору вручную ничего делать не нужно.

## Важные правила

- Для работы с `MCMV2` сохраняйте сценарий как `.xls`.
- Не меняйте структуру Excel вручную, если не уверены.
- Если не понимаете, что такое `0x68`, используйте значения, которые дал инженер.
- Перед передачей файла в `MCMV2` лучше еще раз открыть его и визуально проверить.

## Очень коротко

Если совсем кратко, обычная работа такая:

1. Открыть `WizardScena`.
2. Создать или открыть сценарий.
3. Заполнить вкладки.
4. Сохранить как `.xls`.
5. Открыть этот файл в `MCMV2`.
