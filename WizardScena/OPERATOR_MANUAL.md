# Инструкция для оператора: WizardScena

## Для чего нужна программа

`WizardScena` нужна для подготовки сценария Excel.

Программа умеет:

- создать новый сценарий;
- открыть готовый сценарий;
- исправить параметры сценария;
- сохранить сценарий в Excel;
- подготовить файл для программы `MCMV2`.

Проще всего запомнить так:

- в `WizardScena` сценарий делают;
- в `MCMV2` сценарий запускают.

## Какие файлы использует программа

- открыть можно: `.xls`, `.xlsx`
- для передачи в `MCMV2` сохранять нужно: `.xls`
- эталонный файл графика: `.txt`

Важно:

- не надо вручную переименовывать `.xlsx` в `.xls`;
- нужно сохранять файл именно через `WizardScena`.

## Что видно на стартовом экране

На стартовом экране есть:

- `Открыть сценарий`
- `Создать новый сценарий`
- `Последние сценарии`
- `Настройки`

Список `Последние сценарии` показывает 5 последних открытых файлов.

## Что делают кнопки на стартовом экране

### Открыть сценарий

Открывает уже существующий Excel-файл сценария.

### Создать новый сценарий

Открывает пустой редактор, где можно сделать новый сценарий.

### Последние сценарии

Позволяет быстро открыть один из последних файлов.

### Настройки

Позволяют сменить язык интерфейса:

- Русский
- English

Программа запоминает выбранный язык сама.

## Что делать по шагам

### Создать новый сценарий

1. Запустите `WizardScena`.
2. Нажмите `Создать новый сценарий`.
3. Заполните нужные вкладки.
4. Нажмите `Сохранить Excel`.
5. Введите имя файла.
6. Сохраните файл как `.xls`.

Если имя не введено, программа сама поставит имя `Новый сценарий`.

### Открыть готовый сценарий

1. Запустите `WizardScena`.
2. Нажмите `Открыть сценарий`.
3. Выберите файл `.xls` или `.xlsx`.
4. Подождите, пока файл загрузится.
5. Проверьте содержимое вкладок.

### Открыть один из последних сценариев

1. Запустите `WizardScena`.
2. В блоке `Последние сценарии` нажмите нужный файл.
3. Дождитесь открытия сценария.

### Сохранить сценарий

1. Нажмите `Сохранить Excel`.
2. Выберите папку.
3. Введите имя файла.
4. Убедитесь, что файл сохраняется как `.xls`.
5. Нажмите `Сохранить`.

## Что означают вкладки

### Вкладка `Основной сценарий`

Здесь задаются шаги сценария:

- текст шага;
- длительность;
- время перехода;
- значения каналов;
- `analog 1`;
- `analog 2`.

Основные кнопки:

- `Добавить шаг`
- `Удалить`
- `Копировать выбранные`
- `Вставить до`
- `Вставить после`
- `Выбрать все`
- `Снять выбор`

### Вкладка `Давления`

Здесь редактируются параметры каналов давления:

- отображаемое имя;
- минимум;
- максимум;
- допуск;
- единица измерения;
- видимость канала.

### Вкладка `Токи`

Здесь редактируются параметры соленоидов:

- главная частота;
- отображаемое имя;
- максимальное значение PWM;
- цена деления;
- амплитуда дитера;
- частота дитера;
- минимальный ток;
- максимальный ток;
- видимость канала.

### Вкладка `Параметры 0x68`

Здесь задаются 10 специальных параметров:

- `Параметр 1`
- `Параметр 2`
- ...
- `Параметр 10`

Эти значения программа записывает в Excel в ячейки:

- `D14` ... `M14`

Допустимые значения:

- от `0` до `255`

Если не знаете, что сюда ставить, используйте значения, которые дал инженер.

## Кнопки сверху в редакторе

В редакторе сверху есть кнопки:

- `Выбрать эталон .txt` - выбрать эталонный файл графика
- `Сохранить Excel` - сохранить сценарий
- `Открыть Excel` - открыть другой сценарий
- `Назад на стартовый экран` - вернуться на стартовый экран
- `Настройки` - сменить язык

## Для чего нужен эталонный `.txt`

Эта кнопка нужна, если к сценарию нужно привязать эталонный график.

Простыми словами:

- выбирается `.txt` файл;
- путь к нему записывается в Excel;
- потом `MCMV2` может использовать его при сравнении результатов.

Если эталон не нужен, эту кнопку можно не использовать.

## Типовые алгоритмы работы

### Алгоритм 1. Создать новый сценарий с нуля

1. Открыть `WizardScena`.
2. Нажать `Создать новый сценарий`.
3. На вкладке `Давления` заполнить параметры.
4. На вкладке `Токи` заполнить параметры соленоидов.
5. На вкладке `Параметры 0x68` заполнить 10 параметров, если они нужны.
6. На вкладке `Основной сценарий` добавить шаги.
7. Нажать `Сохранить Excel`.
8. Сохранить файл как `.xls`.
9. Открыть этот файл в `MCMV2`.

### Алгоритм 2. Исправить готовый сценарий

1. Открыть `WizardScena`.
2. Нажать `Открыть сценарий`.
3. Выбрать нужный файл.
4. Изменить нужные вкладки.
5. Нажать `Сохранить Excel`.
6. Сохранить файл снова как `.xls`.

### Алгоритм 3. Подготовить сценарий для MCMV2

1. Открыть или создать сценарий.
2. Проверить вкладки `Давления`.
3. Проверить вкладки `Токи`.
4. Проверить вкладку `Параметры 0x68`.
5. Проверить шаги на вкладке `Основной сценарий`.
6. При необходимости выбрать `Выбрать эталон .txt`.
7. Нажать `Сохранить Excel`.
8. Сохранить как `.xls`.
9. Открыть файл уже в `MCMV2`.

### Алгоритм 4. Быстро открыть недавний сценарий

1. Открыть `WizardScena`.
2. В списке `Последние сценарии` выбрать нужный файл.
3. Проверить содержимое.
4. При необходимости исправить и сохранить заново.

## Частые проблемы

### Сценарий не открывается

Что делать:

1. Проверить, что это Excel-файл.
2. Попробовать открыть другой сценарий.
3. Если файл поврежден, взять резервную копию.

### MCMV2 не принимает сценарий

Что проверить:

1. Файл должен быть сохранен как `.xls`.
2. Параметры `0x68` должны быть в диапазоне `0..255`.
3. Не нужно вручную ломать структуру Excel.
4. После исправлений лучше сохранить файл заново из `WizardScena`.

### Не тот язык интерфейса

Что делать:

1. Нажать `Настройки`.
2. Выбрать нужный язык.

## Где хранятся настройки

Программа сама хранит служебные настройки пользователя:

- выбранный язык;
- список последних сценариев.

Оператору вручную ничего делать не нужно.

## Важные правила

- Для `MCMV2` сохраняйте сценарий как `.xls`.
- Не редактируйте структуру Excel вручную, если не уверены.
- Если не знаете значения `0x68`, спросите инженера.
- Перед передачей файла в `MCMV2` лучше еще раз проверить сценарий глазами.

## Короткая памятка

1. Открыть `WizardScena`.
2. Создать или открыть сценарий.
3. Заполнить вкладки.
4. Сохранить как `.xls`.
5. Открыть этот файл в `MCMV2`.
